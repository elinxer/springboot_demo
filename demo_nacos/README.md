# NACOS安装和配置参考

```
cd /opt/
wget https://github.com/alibaba/nacos/releases/download/1.4.3/nacos-server-1.4.3.tar.gz
tar -zxvf nacos-server-1.4.3.tar.gz
cd nacos
```

```
cd /opt/nacos/bin

# 启动指令 启动命令(standalone代表着单机模式运行，非集群模式):
sh startup.sh -m standalone
```
